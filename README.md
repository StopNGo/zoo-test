# Портал з SMS автентифікацією

## Вимоги

- PHP 8.1 або вище
- Composer
- MySQL 5.7 або вище

## Встановлення

1. Клонуйте репозиторій:
```bash
git clone https://github.com/StopNGo/zoo-test.git
cd zoo-test
```

2. Встановіть PHP залежності:
```bash
composer install
```

3. Скопіюйте файл конфігурації:
```bash
cp .env.example .env
```

4. Налаштуйте файл .env:
- Встановіть параметри підключення до бази даних (DB_*)
- Встановіть APP_KEY:
```bash
php artisan key:generate
```

5. Створіть базу даних та виконайте міграції:
```bash
php artisan migrate
```

## Запуск

1. Запустіть локальний сервер розробки:
```bash
php artisan serve
```

2. Відкрийте у браузері:
```
http://localhost:8000
```

## Особливості проекту

- Автентифікація через SMS (тестовий код: 1234)
- Створення та редагування повідомлень
- Видалення власних повідомлень
- Перегляд всіх повідомлень

## Тестування

Для входу використовуйте:
- Будь-який номер телефону
- Код підтвердження: 1234
